if(i%3==0){age++;}
}
System.out.print(age);

3. Aşağıdakı kodların Java-da icra olunması zamanı name dəyişəninin dəyəri nə olacaq?

int name="Apple";
for(int i=4;i<=20;i+=2){
if(i%3==0){name=name+"a";}
}
System.out.print(name);

b) Yuxarıdakı kod kompilyasiya olunacaq amma, runtime zamanı error baş verəcək
